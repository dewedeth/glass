## For Loops

Writing for loops with Glass can be done with using the `<For>` block. This block will loop over a list of items and render the contents of the block for each item in the list.

<Image src={forloop} />

## Conditional Statements

Writing conditional statements can be done using the `if` attribute on any block. This attribute takes a boolean expression and will only render the block if the expression evaluates to true.
